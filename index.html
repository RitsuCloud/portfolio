Castle Scenery
Castle scenery uses the following tools and technology to create a pleasing autumn scenery with castle on top of cliffs in the distance.
  • Utilizing new nanite rendering technology to decorate the scenery with 8K meshes without worrying about performance.
  • Utilizing Quixel Bridge and Megascans to find high-quality meshes to add to the scenery.
Discord-bot
The Discord Bot is written in Python with Discord API, it contains mini-games like Gamba, Magic 8 ball, and a simple text adventure game. More on the functions of the mini-games below!
  • Gamba: A mini gambling game where the user can register and receive 1000 credits, then the user can use the gamba function which has a 40% chance of winning, or guess a number between 1 to 10 inclusive for a chance to win 10X the amount the user put in, once the user has reach 0 credits the game is over and the user will have to register again to play.
  • Magic 8 Ball: As the name suggested, it functions like a magic 8 ball, ask it a question and it will give the user a respond!
  • Text Adventure: The game will first parse a text file that contains scenes of the game into hashmaps with the key as scene number and value as a “struct”/nameTuple that contains information about the scene. Then the user will try to reach the end of the game without dying by making the right decisions based on the information given. (Still under development)
